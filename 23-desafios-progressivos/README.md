# 23 – Desafios progressivos com temas ODS

Este conjunto de **5 desafios** foi criado para quem já conhece o básico do Streamlit e quer evoluir um app do zero até um projeto pronto para apresentar. Cada desafio acrescenta uma camada nova (páginas, banco de dados, métricas, pitch) e está ligado a um **Objetivo de Desenvolvimento Sustentável (ODS)** da ONU.

## Para quem é

- Alunos que já fizeram os exemplos 01 a 10 (instalação, fundamentos, widgets, multipáginas).
- Equipes do Technovation ou outros programas que pedem um app completo.
- Quem quer treinar do protótipo até um produto apresentável.

## Visão geral dos desafios

| Desafio                                          | Tema ODS         | O que você vai fazer                                                          |
| ------------------------------------------------ | ---------------- | ----------------------------------------------------------------------------- |
| [Desafio 1 – App local](desafio-1-app-local)     | ODS 6 (Água)     | Criar um app em uma página para registrar consumo de água, sem banco de dados |
| [Desafio 2 – Multipágina](desafio-2-multipagina) | ODS 11 (Cidades) | Organizar o app em várias páginas (registro, histórico, dicas)                |
| [Desafio 3 – Supabase](desafio-3-supabase)       | ODS 13 (Clima)   | Conectar ao Supabase e salvar os dados na nuvem                               |
| [Desafio 4 – Métricas](desafio-4-metricas)       | ODS 3 (Saúde)    | Adicionar métricas e gráficos para visualizar evolução                        |
| [Desafio 5 – Pitch](desafio-5-pitch)             | ODS 4 (Educação) | Preparar a apresentação e o roteiro para defender o app                       |

## Ordem recomendada

Os desafios são **progressivos**: cada um usa o que foi feito antes. Faça na ordem 1 → 2 → 3 → 4 → 5.

## Como usar

1. Leia o **Contexto** e os **Requisitos** do desafio.
2. Use os **Exemplos que ajudam** indicados em cada README.
3. Implemente e marque os **Critérios de conclusão**.
4. Opcional: faça a **Sugestão de extensão** para ir além.

## O que você precisa antes

- Python 3.8+ e Streamlit instalados (veja [01-instalacao](../01-instalacao)).
- Ter feito os exemplos [02-fundamentos](../02-fundamentos) até [10-multipaginas](../10-multipaginas).
- Para os desafios 3 em diante: conta no [Supabase](https://supabase.com) (grátis).
