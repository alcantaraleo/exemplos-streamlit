# 04 – Texto e exibição

## Objetivo

Este exemplo ensina as diferentes formas de **mostrar texto** no Streamlit: títulos, subtítulos, markdown, texto simples, legendas e divisórias. Tudo em português e com exemplos coloridos e fáceis de entender.

## Conceitos ensinados

- **st.title** – Título principal da página
- **st.header** e **st.subheader** – Títulos menores para organizar seções
- **st.markdown** – Texto com formatação (negrito, listas, links)
- **st.text** – Texto sem formatação (fonte fixa)
- **st.caption** – Texto pequeno (legenda)
- **st.divider** – Linha horizontal para separar partes
- **st.write** – Mostrar texto, números ou outros dados

## Como executar

```bash
cd 04-texto-e-exibicao
pip install -r requirements.txt
streamlit run app.py
```

- **README.md** – Este arquivo.
